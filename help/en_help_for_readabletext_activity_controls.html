SynPhony Help

Readable Text Controls Section

The Readable Text search function searches through text to find connected text material that is decodable at various stages of learning to read. The search results can be used as examples of text that is readable.

Activity Controls

The Activity Controls section contains a variety of options that let you change the appearance and layout of the activity on the page.

Clicking this button causes the program to search through the available texts for connected text that is readable. Readable text is defined as those words that are decodable based on which letters have been selected.

Text size

Clicking the plus and minus buttons changes the size of the words on the page.

Mininum Words

The default number is 3. This number sets the minimum number of decodable words a text chunk must have in order to be displayed. The text chunk can be a sentence or it can contain multiple sentences.

Items per page

The default value is 50. This number controls how many search results are displayed per page. If there are more than can fit on one page then you can click on the black arrows below the title "Readable Text" on the display page to see the other pages.

Sorting

You can sort the readable text search results in different ways.

  • The Sort by percent orders the text chunks by percentage of readable text. The list is sorted in descending order.
  • Sort by matched words orders the text chunks by the greatest number of matched words. The list is sorted in descending order.
  • Sort by consecutive words orders the text chunks by the greatest number of consecutive decodable words. The list is sorted in descending order.
