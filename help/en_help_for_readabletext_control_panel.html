SynPhony Help

The Control Panel

The entire Control Panel can be hidden and revealed by clicking on the little tab thingy with your mouse. The width of the panel can be changed in the Settings menu which is accessible from the Home page.

Options Section

The Options Section gives you controls for selecting words from the database. You may click on the header text to hide or show the section.

Vocabulary group: 1 2 ...

If the database has multiple groups defined then the Vocabulary group checkboxes appear. They let you select vocabulary from different groups of words.

Syllables: 1 2 ...

These checkboxes control how many syllables the selected words may have. You may select more than one checkbox at a time. It is possible that you may not see any words for some selections you make.

Show only taught letters:

By default this checkbox is selected. If it is selected then SynPhony ensures that all of the displayed words contain only letters that have been taught already. If it is not selected then clicking on a letter will display all words that contain that letter without regard to any other letters in the word.

Show all readable words:

If this checkbox is selected SynPhony finds the words that are composed only of the letters that have been taught already even if they do not contain the focus letter. If this is not selected then SynPhony ensures that each vocabulary word must contain at least one instance of the focus letter for the current lesson. This focussed list finds the words that are useful for the current lesson.

Parts of Speech: nouns verbs...

If the database contains parts of speech information then the Parts of Speech checkboxes appear. They allow you to restrict the word list to words of the selected parts of speech.

Clicking this button clears all the selected letters and allows you to start over again.

Letters Section

The Letters Section lists the different letters or digraphs that exist in the language. You may click on the header text to hide or show the section.

Searching for words

Clicking on the letters will cause SynPhony to search and display words that can be composed completely of letters that have been selected. It is recommended to start by clicking on letters in the top two rows. However, it is not necessary to use the given order for your sequence. If you wish, you may select letters in any order and SynPhony will always try to find words which match the selected letters and display them to you. However, it is possible that when you click a letter that the page does not display any new words. If that happens you can either, 1) add another vocabulary group to the current one or, 2) select an additional syllable length or, 3) select additional letters.

Letter button order

By default the letters are ordered in the most productive order from top left to bottom right. The productive order is one that gives you greatest increase in the number of words each time you introduce a new letter. You may change the order in which the letters appear by clicking on the Settings icon on the Home page, and then specifying a different order in the Custom Sequence field. You must separate each letter with a space.

Letter Page control ◀ 2 ▶

If the language has more letters than can fit on one section then a number and a black triangle will appear in the header that allows you to see the next page. The number of letter buttons that get displayed in the Letters Section can be changed by clicking on the Settings icon on the Home page. The default number of letter buttons in this section is 50.
