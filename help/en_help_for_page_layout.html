The SynPhony interface

The Home page

The Home page is displayed the first time you start the program or by clicking on the Home icon in the top bar.

The Home page contains all of the menu icons in a larger format. It also contains the menu item for the Settings which allows you to customize some aspects of SynPhony.

Page layout

The Top Bar

This area contains the page title and the main menu. It is always visible. The login link lets you sign a person in and allows the computer to keep track of their progress. The main menu contains a number of different icons to access the various features of the program.

  • The Home page contains larger menu icons as well as the Settings page.
  • The Word List page lets you create word lists that only contain the letters you control.
  • The Nonsense Word Generator page lets you create both nonsense and real word lists.
  • The Story Checking Tool page lets you check a sentence or a story to see how readable it is for a lesson.
  • The Word Search page lets you create word search puzzles for each lesson.
  • The Reports page lets you create several different kinds of charts that are unique to the language.

The Control Panel

The Control Panel contains the options used to control what you see in the Activity Area. The content in Control Panel changes depending on the Activity you have selected from the Main Menu. The Control Panel can be hidden by clicking on the little tab thingy that pops out when you hover over it with your mouse. There can be multiple sections in the Control Panel. The headers for each section are collapsible so that you can hide or display them if you need to.

The Activity Area

The Activity Area displays the results of the Control Panel. It often looks like a virtual, white page. The content in the Activity Area depends on the Activity you have selected in the Main Menu. For example, if you are creating word lists, then the white page displays the words which SynPhony has found based on the options you have selected. If you have selected the Word Search page then the Activity Area displays the word search puzzle.

The Activity Controls

The Activity Controls give you additional control over various aspects of the content that gets displayed in the Activity Area. If the content is a word list then the Activity Controls do not change the actual words but only change the appearance or the order of the words on the page.

The Footer

The Footer contains information regarding the program as well as an email link you can use to ask for help.
